## 🌍 About the Organization

**XY.T — Spatio-Temporal Zonal Forecasting** is an open-source research organization dedicated to building scalable, reproducible, and interpretable forecasting systems that operate across both **space** (XY coordinates / geographic zones) and **time** (T).

Our work sits at the intersection of:

| Domain | Focus |
|---|---|
| **Geospatial Analysis** | Zone delineation, spatial aggregation, coordinate-referenced modeling |
| **Time-Series Forecasting** | Temporal dependency modeling, trend & seasonality decomposition |
| **Zonal Statistics** | Area-weighted statistics, zonal summaries, spatial joins |
| **Machine Learning** | Deep learning, gradient boosting, and hybrid spatio-temporal models |
| **Earth Observation** | Remote sensing data pipelines, satellite imagery, raster/vector workflows |

## 🔬 Research Focus

### Spatio-Temporal Zonal Forecasting

Traditional forecasting models operate on a single location or a global aggregate. Our approach models **heterogeneous geographic zones** simultaneously, capturing:

- **Spatial autocorrelation** — nearby zones influence each other
- **Temporal dynamics** — each zone has its own trend, seasonality, and anomalies
- **Cross-zonal interactions** — information propagates across zone boundaries over time

This unlocks applications in:
- Football trajectory prediction
- Position prediction
- Zonal position Predictions

### Dataset

The primary dataset used across this organization's projects is the **SkillCorner Open Data** — 9 matches of broadcast tracking data covering the 2019/2020 top-two finishers in the English Premier League, French L1, Spanish LaLiga, Italian Serie A, and German Bundesliga.

If you use this data in any work built on top of this organization's repositories, please **credit [SkillCorner](https://skillcorner.com)** as the data provider.
